ARCHAIOS SOURCE CODE
===================

The complete analysis pipeline consists of the following components:

1. Data Preparation (Cells 1-5)
   - Raster data processing
   - Feature extraction
   - Known site integration

2. CNN Training (Cell 6)
   - Custom earthwork detection model
   - Multi-scale topographic inputs

3. Belief System & Fusion (Cells 7-8)
   - Multi-evidence integration
   - Water anomaly identification
   - Complex clustering

4. Evidence Generation (Cell 9)
   - Satellite imagery processing
   - Evidence package creation
   - o3 peer review

5. Final Analysis (Cell 10)
   - Result compilation
   - Submission preparation
